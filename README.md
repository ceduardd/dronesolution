## Usage

Create database with SQL scripts contains in `database` folder.

Install [Node.js](nodejs.org).

Download [Last Release](https://github.com/ceduardd/dronesolution/releases).

Install dependencies:

```
npm install
```

Update dependencies:

```
npm update
```

Compile styles:

```
npm run build:css
```

Run:

```
npm run dev
```
