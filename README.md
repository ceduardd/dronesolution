## Description

A web application building with Node.js as backend and OracleXE18c as database engine. This repositorie contain the source code of the client side.

Design building with Tailwind CSS

## Usage

Install dependencies:

```
npm install
```

Update dependencies:

```
npm update
```

Run:

```
npm run dev
```
